# ValentinesDay

A cute Valentine's Day character gallery featuring expressive chibi characters.

## Image Assets

The following character images go in `assets/images/`:

| File | Description |
|------|-------------|
| `flower-love.png` | Chibi character shyly holding a bouquet of red flowers, with hearts |
| `angry-blush.png` | Angry/tsundere chibi character with blushing cheeks and anger mark |
| `starry-eyes.png` | Blushing chibi character with big sparkly eyes, reaching out |

## Usage

Open `index.html` in a browser to view the Valentine's Day gallery page.

Place the three character images in the `assets/images/` directory with the filenames listed above.
